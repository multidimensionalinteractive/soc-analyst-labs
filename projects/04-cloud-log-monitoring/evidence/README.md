# Evidence Folder

Add sanitized proof from the cloud monitoring lab here.

Recommended screenshots:

- CloudTrail or Azure Activity logging enabled.
- Query showing failed login or privilege change.
- Query showing exposure-related resource change.
- Alert details page.
- Teardown or cost-control confirmation.

Do not commit cloud account IDs, tenant IDs, subscription IDs, access keys, ARNs tied to private accounts, or raw logs with sensitive identifiers.
